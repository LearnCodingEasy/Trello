## Vite Press

###### 🖥️ Create Vuepress

```
npm init vuepress trello_vuepress
```

###### 🖥️ Command Path

```
cd trello_vuepress
```

###### 🖥️ Install Sass

```
npm install -D sass-embedded
```

###### 📝 Create File

```
index.md
```

###### 🖥️ Run Vue Press

```
npm run docs:dev
```
