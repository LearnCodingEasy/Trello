# Trello

Trello

## 📦 Run Django

- Modifications To Models File

```cmd
python manage.py makemigrations
```

- Modifications To The Database

```cmd
python manage.py migrate
```

- Run Project

```cmd
python manage.py runserver
```

### 🖥️ Run Vue

```cmd
npm run dev
```

```cmd
npm run build
```

### 🖥️ Run Vue Press

```cmd
npm run docs:dev
```
